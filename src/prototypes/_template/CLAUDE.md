## UI Approach

This project uses **Tailwind CSS v4** and **DaisyUI 5** for styling and components.

- Use DaisyUI component classes (`btn`, `card`, `input`, `modal`, `drawer`, `tabs`, etc.) as the foundation
- Use Tailwind utility classes for layout, spacing, and fine-tuning
- Use `cn()` from `@/lib/utils` for conditional className merging
- Use DaisyUI theme modifiers (`btn-primary`, `btn-ghost`, etc.) for variant styling
- Prefer DaisyUI's built-in semantic classes over custom styles when a matching component exists
